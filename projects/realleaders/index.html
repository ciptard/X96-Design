Real Leaders was fed up with free themes and wanted a custom blog design.

About the Project

Real Leaders is a blog covering all topics surrounding leadership including trust, communication, and more. The goal of this theme was to highlight the post images (using featured post thumbnails), and to integrate more colour than the last theme (a minimalist/wireframe theme).

Details

Format: WordPress

Features: Featured post thumbnails, two sidebars, latest post featured on homepage.
