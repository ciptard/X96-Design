Howdy. I'm a young, aspiring web nerd on the west coast of Canada. My passion is programming and designing, and I've been doing just that since the age of eight.

The History

My dad bought a book for himself on HTML because he wanted to build a family website for us. A couple years afterward, I found the book and my dad said I could have it.

I read that book cover-to-cover too many times to count. I memorized everything the book had to offer.

Originally, I stated on the only computer I had, an old Windows 95 Compaq that my parents got on sale at some computer repair shop. My first website was an Intranet site for my secret spy company (I was 8).

From there, I got to where I am now: writing much better code, designing websites using current technology and specs, and, best of all, continuing to pursue what I love doing.

About this site

This site is running on pure, static HTML. No CMS, no backend coding, just clean, simple static HTML files.

I'm using Apache as a web server, but I prefer NGINX (long story). The server is a Rackspace Cloud Server running Ubuntu 11.04.

The site will look decent in all the web browsers that actually work (Firefox, Chrome, Safari). I've done basic testing in Internet Explorer (just enough so it'll somewhat resemble what it's supposed to look like). Opera I don't care about (and, quite frankly, I'm not sure if anyone does).
